Project Description

ExchangeEase is a comprehensive "Survival Dashboard" designed specifically for international exchange students. It provides real-time essential dataâ€”including currency conversion, emergency contacts, local transit updates, and a community-driven gig/guide networkâ€”all in one centralized, location-aware interface.

The Problem statement

Moving to a new country for studies is overwhelming. Students often struggle to find reliable local information, manage different currencies, understand public transport, and connect with seniors or locals who can help with visa or housing issues, leading to "culture shock" and safety concerns.

The Solution

ExchangeEase solves this by aggregating "survival essentials" into a single glassmorphic dashboard. It uses real-time maps and location-based data to provide immediate access to emergency services, financial tools, and a peer-to-peer network for jobs and mentorship.

Features

List the key features of your project:

Dynamic Hub Selection: Automatically updates currency, emergency numbers, and transit data based on the searched city (Berlin, Delhi, Paris, London).

Interactive Survival Map: A custom-styled map for orientation and local search.

Real-time Currency Converter: Instant conversion from USD to local currency with live rate displays.

Community Feed: A peer-to-peer system where students can post part-time "gigs" or offer services as local "guides."

One-Tap SOS: An emergency alert button designed for quick access to local authorities.
